# financial-complaint-nlp-genai-assistant

## Data Setup

This project uses a Kaggle dataset. Data files are **not included in the repo**.

Download the dataset from Kaggle and place it in the project root (or a `data/` folder):

- **Dataset:** [Consumer Complaints Dataset for NLP](https://www.kaggle.com/datasets/shashwatwork/consume-complaints-dataset-fo-nlp)
- **File used:** `complaints_processed.csv`

Data files are listed in `.gitignore` and should never be committed.
